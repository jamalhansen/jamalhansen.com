## Why This Caught My Eye

Minimal, text-first, no ads, no tracking. An interesting counterpoint to the complexity of modern web infrastructure. We built the web to be "free", which evolved it into being very useful adware that tracks you. 

When I first found the internet in college, the HTTP protocol was just being adopted. FTP and Gopher were still around. I believed at the time that we would see many more protocols as the web grew and evolved, and we haven't yet.
